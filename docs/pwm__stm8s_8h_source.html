40 /*
41  * Throttle control servo signal timings are measured with th Spektrum DX8
42  * transmitter and AR620 6-channel air receiver. With the defailt frame rate
43  * setting, the rate is observed to be:
44  *
45  * The timer must be a 16-bit timer with the period set to maximum 0xFFFF and
46  * remains as a free running timer with a maximum time of:
47  *
48  * 1/16Mhz * prescaler * 0xFFFF = 4.1ms * 8 == == 0.0327675 (32 ms)
49  * 1/16Mhz * prescaler = 1/16Mhz * 8 = 0.0000005 = 0.5uS/tick
50  *
51  * The pulse period is measured on the scope at about 22 ms. Software observes
52  * a count of $AC80 (44160d), converting to time:
53  *
54  * 44160 * 0.5uS/tick = 22.08 ms
55  * 1/22.08 ms = ~45Hz
56  *
57  * Reference:
58  * https://www.kdedirect.com/blogs/news/understanding-throttle-calibration-esc-deadbands-and-pwm
59  *
60  * Motor Arming pulse: 1100탎
61  * Motor Spinning pulse: 1125탎
62  * Motor Stopping pulse: 1110탎
63  * Max thrust output: 1915탎
64  * Full Stick: 1940탎
65  */
