converts servo pulse width to motor speed percent

Commanded motor speed is derived from proportional servo pulse so it is converted to percent of throttle/speed range.

Parameters

pulse_duration_countsservo pulse duration in timer counts

Converts servo pulse width to servo position.

100% of servo throttle range resides in the portion of the servo pulse i.e. (1.1 ms : 1.9 ms) i.e. servo position = servo pulse time - 1.1 ms

Returns
duration of servo pulse expressed as timer counts, range (0:1600)
