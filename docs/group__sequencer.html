◆ Sequence_Step_0()

void Sequence_Step_0 ( void  )

Public accessor for step 0 in the commutation sequence function table.

The sequence is initialized by the Alignment to Ramp transition. The Alignment sets the Sector to 0 and waits some time for the motor to align. After alignment, the control state machine enables the sequence which will be pointed to Sector 1 as the next step.
